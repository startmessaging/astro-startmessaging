El envío de notificaciones y ofertas comerciales masivas a través del WhatsApp es uno de los canales con mejor retorno de inversión en el mercado mexicano. Con tasas de lectura inmediatas, las campañas en este medio permiten a las marcas de retail, servicios y fintech conectar con sus clientes de forma directa. No obstante, el envío de contenido no solicitado o el uso de plataformas irregulares provoca el bloqueo permanente del número de teléfono comercial por parte de Meta. Entender las reglas vigentes para enviar whatsapp masivo mexico de forma oficial (WhatsApp Business API Broadcast) y aplicar buenas prácticas de frecuencia y consentimiento es fundamental para proteger tu línea y asegurar la efectividad de tus campañas de mercadotecnia.

La API del StartMessaging para WhatsApp se encuentra actualmente en fase de desarrollo por nuestro equipo técnico. Mientras se liberan los endpoints oficiales del entorno de producción, los cuales se liberarán próximamente, ya puedes crear tu lógica de distribución masiva y enviar tus campañas de prueba utilizando la ruta temporal `/v1/messages`. El acceso y la autorización se validan de forma segura por medio del encabezado HTTP personalizado `X-API-Key`.

## Envíos Oficiales via API vs. Herramientas de Spam no Oficiales

Muchas marcas en busca de soluciones económicas recurren a softwares piratas, scripts locales o extensiones de navegador para realizar envíos en masa a bases de datos. Estas herramientas simulan el uso de la aplicación móvil de WhatsApp enviando mensajes de texto repetitivos a través de WhatsApp Web.

El uso de estas aplicaciones informales representa un gran riesgo operativo para tu empresa. El algoritmo de Meta monitorea la velocidad de envío, la cantidad de destinatarios nuevos que no tienen tu contacto guardado y el porcentaje de reportes de spam. Cuando detecta estos patrones de comportamiento robótico, Meta suspende la cuenta y el número telefónico de forma definitiva en cuestión de minutos, imposibilitando la recuperación de tus conversaciones de clientes.

La única alternativa legal, segura y autorizada para realizar envíos masivos comerciales en México es el canal del WhatsApp Business API (WhatsApp Broadcast). Al estar conectado directamente con los servidores de nube de Meta, este canal te permite enviar miles de notificaciones promocionales homologadas de manera segura y garantizando la entrega de tus mensajes directamente en la bandeja de entrada del usuario.

## El Semáforo de Calidad de Número (Quality Rating) de Meta

La Meta utiliza un sistema de reputación para controlar la calidad del contenido enviado a través del WhatsApp Business API, con el fin de evitar que el canal corporativo se llene de mensajes indeseados. Este indicador, conocido como **Quality Rating** (Calidad de Número), se actualiza constantemente en el administrador comercial de Meta Business Suite.

El estado de calidad de tu línea comercial se clasifica bajo tres colores de reputación:

- **Verde (Calidad Alta):** El número de teléfono cumple con las políticas comerciales de Meta, los clientes muestran interés en las comunicaciones y el índice de reportes de spam o bloqueos por mensaje enviado es extremadamente bajo.
- **Amarillo (Calidad Media):** Se identifica un incremento en el porcentaje de destinatarios que eliminan tus chats sin responder o los marcan como publicidad molesta. Es un indicador claro para segmentar mejor tus listas de contactos y optimizar el contenido.
- **Rojo (Calidad Baja):** Tu número comercial ha superado la tasa permitida de quejas y bloqueos de usuarios. Al caer en esta categoría, Meta puede restringir el límite diario de envíos permitidos para tu número, reducir tu nivel de escalabilidad o suspender temporalmente tu cuenta publicitaria comercial.

## Modelos de Mensajes Homologados (Templates): Disparos y Código

Para iniciar conversaciones comerciales con usuarios (mensajes activos), la API del WhatsApp Business requiere que se utilicen modelos de mensajes pre-aprobados (Message Templates) validados por Meta. Esto garantiza que la información promocional se mantenga dentro de las políticas de comercio de la plataforma.

A continuación se muestra el código de Node.js que realiza un envío masivo de template promocional utilizando Axios y el endpoint provisional de StartMessaging:

```javascript
const axios = require('axios');

/**
 * Dispara una plantilla de mensaje promocional (Marketing) a un cliente registrado en México.
 * 
 * @param {string} toClientPhone - Celular del destinatario con código de país (ex: "525598765432")
 * @param {string} nameClient - Nombre personalizado del comprador
 * @param {string} codeCoupon - Cupón de descuento dinámico generado para el cliente
 */
async function dispararPlantillaPromocional(toClientPhone, nameClient, codeCoupon) {
  // TODO: Actualizar al endpoint oficial de producción tan pronto como el soporte del WhatsApp esté liberado
  const apiEndpointUrl = 'https://api.startmessaging.com/v1/messages';

  const payload = {
    to: toClientPhone,
    type: 'template',
    template: {
      name: 'buen_fin_descuentos',
      language: {
        code: 'es_MX'
      },
      components: [
        {
          type: 'body',
          parameters: [
            { type: 'text', text: nameClient },
            { type: 'text', text: codeCoupon }
          ]
        }
      ]
    }
  };

  try {
    const response = await axios.post(apiEndpointUrl, payload, {
      headers: {
        'X-API-Key': process.env.STARTMESSAGING_API_KEY,
        'Content-Type': 'application/json'
      }
    });

    console.log(`Mensaje de campaña enviado con éxito para ${nameClient}. ID: ${response.data.messageId}`);
    return response.data;
  } catch (error) {
    console.error('Error al enviar la campaña de broadcast por WhatsApp:', error.response ? error.response.data : error.message);
    throw error;
  }
}

// Ejemplo de disparo unitario simulando el recorrido de un bucle de contactos de tu base
const celularDestino = '525598765432';
const nombreUsuario = 'Sofía Ramírez';
const cuponDesconto = 'BUENFIN10';

dispararPlantillaPromocional(celularDestino, nombreUsuario, cuponDesconto);
```

Este script de backend envía el payload correspondiente especificando el template a usar y sus variables, ejecutando la llamada mediante la autorización con la clave secreta `X-API-Key`.

## Cómo Construir una Base de Contactos con Consentimiento (Opt-in)

La base de un canal de WhatsApp Marketing exitoso y de alta conversión reside en la calidad de tus listas de contactos. De acuerdo con la ley mexicana de protección de datos (LFPDPPP), la compra de bases de datos externas o la extracción masiva de directorios públicos está prohibida y expone a las empresas a multas graves del INAI.

Las principales metodologías para captar de forma orgánica la autorización de tus clientes son:

- **Casilla de Selección en Checkout (Carrito de Compra):** Añade un selector específico e independiente en tu formulario de compra web (por ejemplo: "Deseo recibir ofertas y actualizaciones en mi número de WhatsApp").
- **Estrategia de Código QR en Tienda Física:** Coloca displays con códigos QR en el mostrador o mesas de tus sucursales ofreciendo un beneficio inmediato (como un descuento de $100 MXN en su compra) al iniciar una conversación en WhatsApp.
- **Formularios de Registro en Landing Pages:** Invita a tus visitantes a unirse a un club de preventas exclusivas a cambio de ingresar de forma voluntaria su número celular en tu sitio web.

## Frecuencia, Horarios Recomendados y Uso de Opt-out en México

El consumidor mexicano en WhatsApp reacciona de forma favorable a las ofertas de valor, pero rechaza los mensajes recurrentes e intrusivos. Definir reglas claras de frecuencia y horario en tus sistemas de automatización protegerá la vida útil de tu número de atención.

Las buenas prácticas comerciales de envío masivo son:

- **Respetar los Horarios de Descanso:** No realices envíos masivos comerciales antes de las 10:00 AM ni después de las 7:00 PM. Mensajes recibidos en la noche o los fines de semana suelen generar mayores tasas de quejas y reportes por spam.
- **Moderar la Frecuencia de Envío:** Limita tus campañas generales a un máximo de 1 o 2 envíos promocionales por semana para un mismo destinatario.
- **Inserción Obligatoria de Opt-out:** Todo mensaje promocional debe finalizar ofreciendo una vía de salida clara para el cliente (por ejemplo: "Si prefieres no recibir más ofertas, responde con CANCELAR"). Tu sistema debe capturar esta respuesta a través de webhooks y remover de inmediato el contacto de las listas activas de disparo para cumplir con las regulaciones de la LFPDPPP.

## Preguntas Frecuentes

**Q: ¿Cuánto cobra Meta por el envío de mensajes promocionales en México?**

A: Meta aplica un esquema de tarifas basado en conversaciones de 24 horas y el costo varía según la categoría del template. En México, las conversaciones de la categoría de "Marketing" (promociones) son las que tienen la tarifa más alta de la plataforma y su valor promedio oscila entre $0.60 y $0.80 MXN por sesión iniciada por el negocio.

**Q: ¿Puedo realizar disparos masivos utilizando números locales tradicionales de 10 dígitos?**

A: Sí. La API de WhatsApp empresarial admite el registro de líneas de 10 dígitos (números fijos y celulares convencionales de México). Durante el proceso de validación en la consola de Meta, deberás seleccionar la recepción del código de verificación mediante llamada telefónica o SMS.

**Q: ¿Existe un número máximo de personas a las que puedo enviar un broadcast por día?**

A: Sí, de acuerdo con los límites diarios de envío (sending limits) administrados de forma automática por Meta. Las cuentas de WhatsApp Business nuevas inician en el Nivel 1, con un límite de envío a 1,000 clientes únicos en un periodo de 24 horas. Este límite incrementa automáticamente a 10,000, 100,000 e ilimitado conforme realices disparos de alta calidad.

**Q: ¿Qué pasa si muchos usuarios reportan mis campañas de WhatsApp masivas como spam?**

A: El estatus de calidad de tu número telefónico (Quality Rating) caerá rápidamente de Verde a Amarillo y finalmente a Rojo. Si tu número de teléfono permanece en calidad roja por mucho tiempo, Meta puede degradar tus límites de mensajería al nivel inferior, bloquear temporalmente el envío de plantillas nuevas o suspender la cuenta publicitaria comercial de tu marca.
